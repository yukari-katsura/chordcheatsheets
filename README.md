# ChordCheatSheets

固定ドのドレミ表記とピアノ鍵盤のレイアウトを対応させた、各調ごとのコード・チートシートです。  
C major / A minor をテンプレートにして Python で自動生成した HTML を、GitHub Pages で公開しています。

## 目的

- コードネームを見たときに、すぐに頭の中で鍵盤上の形としてイメージできるようにする
- 「その調の中でよく出てくるコード」を一覧として常に手元に置けるようにする
- すべてのコードを暗記するのではなく、調ごとのチートシートを参照しながら演奏・分析できるようにする

## 構成

このリポジトリには、次のようなファイルが含まれています。

- `index.html`  
  全調のチートシートを切り替えて表示するインデックスページです。  
  左側に調の一覧があり、クリックすると右側のフレーム内で該当調の HTML が表示されます。

- `Cmajor.html`, `Gmajor.html`, `Dmajor.html`, ...  
  各メジャーキーごとのチートシートです。  
  例:
  - C, G, D, A, E, B, F♯, C♯ (シャープ系)
  - F, B♭, E♭, A♭, D♭, G♭ (フラット系)

## 表示仕様の概要

各 HTML では、次のようなレイアウトでコードが表示されます。

- 左端の列: コード名
  - 代表名 + 別名を改行で併記  
    - 例: `Cmaj7` と `CM7`  
    - 例: `GonB` と `G/B`
- 右側: 2オクターブ分の鍵盤を左右方向に展開した表
  - 上段: 黒鍵  
    - 背景色: 黒、文字色: 白  
    - 黒鍵のない位置では、2セルを `colspan` で結合して空白にしています
  - 下段: 白鍵  
    - 背景色: 白、押されていないときは文字色も白（実質的に空白）
    - コード構成音がある鍵盤のみ、ドレミ（固定ド）を表示します
    - ルート音は太字で表示されます

### 固定ド表記とキーごとの例外

基本的には、白鍵は次のように固定ドで表示します。

- C: ド
- D: レ
- E: ミ
- F: ﾌｧ
- G: ソ
- A: ラ
- B: シ

ただし、以下のキーでは聴感に合わせて一部の音名をずらして表示しています。

- F♯major / C♯major
  - C → シ
  - F → ミ

- G♭major
  - B → ド

黒鍵のドレミは、調に応じて「シャープ解釈」「フラット解釈」を切り替えています。
